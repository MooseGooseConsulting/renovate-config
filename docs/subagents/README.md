# Subagent Research Archive

This directory contains static reports written by delegated agents during the
Renovate strategy pass. These files are evidence leads and critique notes. They
are not runnable subagents, not policy authority, and not a substitute for
current official Renovate documentation.

Promote an idea from these reports only after it passes the North Star,
architecture invariants, current Renovate docs, validation, and consumer-repo
evidence.

| Report | Use |
| --- | --- |
| `config-inventory.md` | Local snapshot of current config surfaces and tensions. |
| `renovate-official-research.md` | Concise official-doc research notes from the documentation pass. |
| `renovate-change-agent-critique.md` | Independent critique of using Renovate as an agent-governed change surface. |
| `skills-research.md` | Local machine skill inventory and provenance notes; treat as quickly stale. |
| `exceptional-configs-and-ai-patterns.md` | Raw inspiration report; useful idea mine, but contains claims that require verification before promotion. |
| `renovate-combinator-critique.md` | Critique of the detailed combinator/dead-dependency proposal and its evidence boundaries. |
